# weatherAppWithOpenWeatherMapAPI
📱 OpenWeatherMap의 API를 이용해서 간단한 날씨앱을 구현해보는 프로젝트입니다. <br>
피그마를 이용해서 디자인을 진행해보고 청년취업사관학교 영등포 캠퍼스 기준의 데이터만 받아와서 적용해보았습니다.
